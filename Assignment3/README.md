## Description
This project implements a class hierarchy of "Transformers". 
- **Base Class**: `Transformer`
- **Derived Classes**: `Autobot`, `Decepticon`, `MiniCon`
- **Features**: Class composition (`Weapon`), association (`Pilot`), and inheritance.
